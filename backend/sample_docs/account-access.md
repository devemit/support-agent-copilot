# Account Access

For login problems, agents should first confirm whether the customer is using the correct email address and authentication method. Password reset links expire after 30 minutes.

If a customer cannot receive a password reset email, agents should ask the customer to check spam folders, confirm the email spelling, and verify whether the account uses single sign-on.

Accounts locked after repeated failed login attempts automatically unlock after 15 minutes. Agents may escalate persistent lockouts to the security team.
